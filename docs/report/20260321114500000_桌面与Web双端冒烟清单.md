# 桌面与 Web 双端冒烟清单

## 适用范围
- 前端页面与交互改动
- Web 运行壳、`/api/runtime`、`/api/rpc` 相关改动
- 桌面端与 Web 端共用页面的兼容性改动

## 自动化预检查

### 前端静态导出
```bash
pnpm -C apps run build
```

预期：
- 构建通过
- `/`、`/accounts`、`/apikeys`、`/logs`、`/settings` 都能生成静态页面

### Web 运行壳单测
```bash
cargo test -p codexmanager-web
```

预期：
- `/api/runtime` 相关测试通过
- Web 登录页、地址归一化相关测试通过

### Web 运行壳离线脚本测试
```powershell
pwsh -NoLogo -NoProfile -File scripts/tests/web_runtime_probe.test.ps1
```

预期：
- 输出 `web_runtime_probe.ps1 synthetic runtime check looks ok`

### Web 页面级 smoke
```powershell
pwsh -NoLogo -NoProfile -File scripts/tests/web_ui_smoke.ps1 -SkipBuild
```

预期：
- supported mock 运行壳下，账号页、密钥页、日志页、访问密码弹窗都能正常打开
- unsupported mock 运行壳下，会直接显示“当前 Web 运行方式不受支持”
- 脚本执行结束后不会在仓库根目录留下 `.playwright-cli` 等临时产物

### 真实 codexmanager-web 页面 smoke
```powershell
pwsh -NoLogo -NoProfile -File scripts/tests/web_shell_smoke.ps1 `
  -SkipFrontendBuild -SkipRustBuild
```

预期：
- 在隔离数据目录里自动拉起 `codexmanager-service` 与 `codexmanager-web`
- `账号管理`、`平台密钥`、`请求日志`、`应用设置` 页面都能正常打开
- 访问密码弹窗可打开
- 更新区域能显示 Web 降级提示，不会误暴露桌面更新能力

## Web 端手工冒烟

### 1. 运行方式确认
启动 `codexmanager-web` 后，执行：

```powershell
pwsh -NoLogo -NoProfile -File scripts/tests/web_runtime_probe.ps1 `
  -Base http://localhost:48761
```

预期：
- `Mode` 为 `web-gateway`
- `RpcUrl` 指向 `http://localhost:48761/api/rpc`
- `ServiceName` 为 `codexmanager-service`

### 2. 仪表盘启动
浏览器打开 `http://localhost:48761/`

预期：
- 页面能正常打开
- 不出现“当前页面缺少 CodexManager Web 运行壳”错误
- 不出现“无法同步核心服务状态”误报

### 3. 账号页
进入“账号管理”

预期：
- 列表可以正常加载
- “选择文件导入”“选择目录导入”“导出到浏览器”文案符合 Web 语义
- 导出后 toast 不再出现 `browser-download`

### 4. 平台密钥页
进入“平台密钥”

预期：
- 列表加载正常
- 可创建、编辑、删除
- 可读取密钥明文

### 5. 日志页
进入“请求日志”

预期：
- 列表和汇总加载正常
- 清空日志操作正常

### 6. 设置页
进入“设置”

预期：
- Web 密码可以设置和清除
- 更新相关按钮处于正确的 Web 降级状态
- 托盘相关开关不会在 Web 端表现成可执行能力

### 7. unsupported-web 校验
仅托管前端静态文件，不提供 `/api/runtime` 或 `/api/rpc`

预期：
- 页面直接提示当前运行方式不受支持
- 提示通过 `codexmanager-web` 打开或补齐代理
- 不再把错误归因成 service 初始化失败

## 桌面端手工冒烟

### 1. 应用启动
启动桌面端

预期：
- 正常进入首页
- Header 可见端口与服务启停控件

### 2. 服务启停
在 Header 中启动与停止服务

预期：
- 启动成功后，状态变为“服务已连接”
- 停止后回到“服务未连接”
- 不影响页面导航

### 3. 账号页
进入“账号管理”

预期：
- “按文件导入”“按文件夹导入”“导出账号”文案仍保持桌面语义
- 刷新账号用量、导入、导出正常

### 4. 设置页
进入“设置”

预期：
- 更新按钮按桌面能力正常工作
- 托盘开关行为正常
- 打开日志目录或本地目录能力不受影响

## 记录要求
- 成功项记为“已验证”
- 因环境限制无法验证的项，明确写“未验证 + 原因”
- 若发现问题，至少记录：
  - 页面
  - 操作步骤
  - 实际结果
  - 预期结果
  - 所在运行方式（桌面 / Web / unsupported-web）
