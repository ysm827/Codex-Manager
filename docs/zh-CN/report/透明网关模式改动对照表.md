# 透明网关模式改动对照表

> 临时文档  
> 用途：快速审阅当前分支已经落地的“Codex 原生直通 + Claude/Gemini 协议适配”改动。

## 1. 总览

| 模块 | 改动项 | 当前结果 |
| --- | --- | --- |
| 产品语义 | 不再暴露“透明 / 增强”切换 | 固定为 `Codex` 原生直通，`Claude/Gemini` 继续走协议适配 |
| 后端设置接口 | 移除 `gatewayMode` 设置读写 | 前后端都不再能切换模式 |
| 前端设置页 | 移除“网关模式”选择器 | 设置页只保留与路由、账号池、请求头相关的固定配置 |
| 前端日志页 | 移除模式徽标和模式说明 | 日志只展示请求类型、路由路径、上游信息等有效观测字段 |
| 运行时行为 | 默认保持 `Codex` 原生请求最小改动 | 账号替换、路由、会话亲和、日志保留；不再让用户理解成“增强模式” |
| 协议兼容 | 保留 `Claude/Gemini` 适配链路 | 不影响 `anthropic_native / gemini_native` 的入口能力 |
